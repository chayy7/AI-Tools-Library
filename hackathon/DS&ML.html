Data Science & Machine Learning

TensorFlow

TensorFlow is an open-source machine learning library developed by Google for various machine learning tasks, including neural network training.

PyTorch

PyTorch is an open-source deep learning platform known for its flexibility and speed, widely used for research and production in AI.

Scikit-learn

Scikit-learn is a Python library for simple and efficient tools for data mining and data analysis, built on NumPy, SciPy, and Matplotlib.

Keras

Keras is a high-level neural networks API, written in Python, capable of running on top of TensorFlow, CNTK, or Theano.

XGBoost

XGBoost is an open-source library that provides an optimized distributed gradient boosting framework designed for high performance and efficiency.

Apache Spark (MLlib)

MLlib is Apache Spark's scalable machine learning library that provides various algorithms for classification, regression, clustering, and more.

LightGBM

LightGBM is a gradient boosting framework that uses tree-based learning algorithms, designed for efficiency and scalability in handling large datasets.

H2O.ai

H2O.ai provides open-source software for scalable machine learning and data analytics, offering powerful tools for building predictive models.

CatBoost

CatBoost is a gradient boosting library that handles categorical features directly, aiming to reduce the need for extensive data preprocessing.

KNIME

KNIME is an open-source data analytics, reporting, and integration platform that supports various machine learning and data mining tasks.

Weka

Weka is a collection of machine learning algorithms for data mining tasks, with tools for data preprocessing, classification, regression, clustering, and visualization.

Matplotlib

Matplotlib is a Python plotting library that provides an object-oriented API for embedding plots into applications and interactive data visualization.

Seaborn

Seaborn is a Python visualization library based on Matplotlib that provides a high-level interface for drawing attractive and informative statistical graphics.

Tableau Public

Tableau Public is a free version of Tableau that allows users to create and share interactive and visually appealing data visualizations online.

BigML

BigML offers a free tier for its machine learning platform, providing tools for creating, evaluating, and deploying machine learning models easily.
